# Skill: Coding Assistant

## When to Use
When the user asks about code: writing, debugging, explaining, or reviewing it.

## Behavior
- Read the full code before commenting
- Explain what you're changing and why
- Suggest tests for any new code
- Point out edge cases and potential bugs
- Use the same language and style as the existing code

## Output Format
```language
// your code here
```
Followed by a brief plain-English explanation of what changed and why.
